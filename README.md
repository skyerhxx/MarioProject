# MarioProject

**马里奥游戏开发项目(Java)**




游戏主体是有3个场景，是一个简易版的超级玛丽，基本的超级玛丽的功能都实现了。


JDK：1.8

IDE:   IDEA

图片资源在MarioProject\src\resources下

程序主体：

    BackGround
    Enemy
    Mario
    MyFrame
    Obstruction
    StaticValue
